Behind Story##

The most challenge part was 'Saving search result values by each input row' on user click in order to add new trip form when both values of the input row's filled out.

For instance, to get 'Trip1: destination' value when saving 'Trip1: departure' value, I first needed to check if 'Trip1' object is exist in Redux store. If not, set the value as empty.

It's solved by using normalized data structure, creating an array of 'Trip Id's. so I can quickly check if 'Trip1' object is Exist.
